# 🎮 Mini PUBG 2D Game (Python + Pygame)

A simple PUBG-style 2D survival shooting game built using Python and Pygame.

Players can:
- Move using WASD
- Aim with mouse
- Shoot enemies
- Survive as long as possible
- Increase score by defeating enemies

---

# 🚀 Features

✅ Smooth player movement  
✅ Mouse aiming system  
✅ Shooting mechanics  
✅ Enemy AI chasing player  
✅ Health system  
✅ Score tracking  
✅ Game over screen  
✅ Random enemy spawning  

---

# 🛠️ Technologies Used

- Python 3
- Pygame

---

# 📦 Installation

## 1. Clone Repository

```bash
git clone https://github.com/HarishchandraChaudhary/mini-pubg-python.git
```

---

## 2. Open Project Folder

```bash
cd mini-pubg-python
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Game

```bash
python main.py
```

---

# 🎮 Controls

| Key | Action |
|------|--------|
| W | Move Up |
| S | Move Down |
| A | Move Left |
| D | Move Right |
| Mouse | Aim |
| Left Click | Shoot |
